Install software
```powershell
choco install adobereader -y
```
List installed software
```powershell
choco list
```
Uninstall software
```powershell
choco uninstall adobereader -y
```
Forcing software installation
```powershell
choco install adobereader -y -force
```
Search for package
```powershell
choco search adobereader
```
